# CodeAlpha_BasicNetworkSniffer
This project is a Basic Network Sniffer developed using Python to analyze network traffic. It captures and displays key packet details such as source IP , destination IP address, protocol type, and payload data.The project helps to understand network communication, packet structure, and the fundamentals of network monitoring and traffic analysis.

Features

* Monitors network traffic
* Displays source and destination IP addresses
* Identifies network protocols
* Shows packet payload information
* Helps understand packet flow and network communication
  
Technologies Used

* Python
* Networking Concepts
* Packet Analysis Techniques

Applications

* Network Monitoring
* Traffic Analysis
* Educational and Research Purposes
* Cybersecurity Learning

Conclusion

This project provides a simple introduction to network packet analysis and monitoring. It helps users understand how data travels across a network and builds a strong foundation for advanced networking and cybersecurity studies.
